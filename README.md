# Signup Page Design in Figma

## Project Overview
This project is a modern and user-friendly Signup Page designed using Figma. The design focuses on simplicity, accessibility, and a clean user interface.

## Features
- Modern UI design
- Full Name input field
- Email input field
- Password input field
- Confirm Password field
- Sign Up button
- Social Login options
- Login link for existing users
- Responsive layout concept

## Tools Used
- Figma

## Design Elements
- Clean typography
- Rounded input fields
- Soft color palette
- Consistent spacing and alignment
- User-friendly form layout

## Learning Outcomes
Through this project, I learned:
- Creating frames and layouts in Figma
- Using Auto Layout
- Working with typography and colors
- Designing responsive user interfaces
- Building modern authentication screens
